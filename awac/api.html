AWAC: Android Web App Container

API for AWAC

Awac introduces an object representing the container which can be taled to.

new Awac()
var awac = new Awac([varname])

Create an object representing the container. If the object is going to be used to receive events then the name of the variable should be passsed as a parameter.

endPage()
awac.endPage([result])

Tell the container to pop the current page off the stack. Optionally the page can return a result parameter to the parent page. This may be a Jabascript primitive, a JSON object or a JSON array.

openPage()
awac.openPage(pageTag,pageUrl,[initParam])

Open a new page and push it onto the stack. The contents of the page is identified by a url that may be absolute or relative. The page tag is used to identify the page if it returns a result using endPage(). Optionally an'init parameter' may be passedto the page page. This may be a Javascript primitive, aJSON object or a JSON array.

startPage()
awac.startPage()

Inform the container that the page is ready to be displayed. This is typically handled by the pages on load event handler.

setOnPageClose()
awac.setOnPageClose(function(pageTag,ok,[result]))

Specify a function to be called when a child page created by openPage() is closed. The pageTag can be used to identify the child page. The ok parameter indicates if the page was explicitly ended or whether the default back behaviour was responsible for ending the page. If endPage() was called with a result then it is passed back.
