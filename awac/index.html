What is AWAC?

There is a certain class of app where the content is key - for example live dynamic content. If you just want to see what that might look like in an app and you have basic web skills then Awac makes it easy to build a working app prototype.

Write once and deploy to multiple platforms

Or at least that's the plan, but we're not there yet...

Currently the only web-app container we have is Awac which supports Android. But ideally we would also have an Iwac (iOS Web App Container) and at least one web-based container. The intent is that it should be possible for the same apps to run on multiple platforms.
