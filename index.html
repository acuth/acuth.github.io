Android Web App Container

Welcome to the Android Web App Container project. Currently Awac is a pair of repositories designed to provide an easy way to prototype Android aplication using web tecnologies:

  • acuth/Awac - an Android app that acts as a container for web apps
  • acuth.github.io - a set of web pages (including this one) that provide some sample web apps

A prebuilt version of the Awac APK is available at awac.apk

By default, the prebuilt Awac APK points to the web app at https://acuth.github.io/test/index.html
