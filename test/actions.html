In this example we create a global variable to represent Awac called _awac. Actions are defined by a 'label' and an 'action string'.
       _awac = new Awac('_awac');
       _awac.setOnAction(function(action) {...});
       
       _awac.addActionBarItem(
          {'label':'OK','action':'ok'});
       _awac.addOptionsMenuItem(
          {'label':'Settings','action':'settings'});
       
       _awac.addNavDrawerItem(
          {'label':'Close','action':'exit'});
       _awac.unlockNavDrawer();
The nav drawer is only populated once for the app and, by default, is locked close for a page.

In Awac an 'action' is a general term for an event generated by menu item which has an associated 'action string'. Menu items can be found in the nav drawer, the action bar and the options menu.

Note that, for this page, the nav drawer only generates a log entry (see below).
