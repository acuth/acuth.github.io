Awac allows a background task to be run which the current page can make requests to.
_awac = new Awac('_awac'); ... _awac.startBackground('background.html'); ... _awac.getBackground('status',function(result) { });

Awac allows an app to define a single long-running background task that can be communicated with by the current page. The background task is defined by an html file that is never made visible.
