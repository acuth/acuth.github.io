In this example we create a global variable to represent Awac called _awac.
   _awac = new Awac('_awac');
   
   var positive = 'red';
   var negative = 'blue';
   _awac.dialog('Which pill are you going to take?',
     positive,negative,
     function(positiveClick){...});
   
   
   _awac.alert('This is what an alert looks like');

Awac provides access to native alerts, dialogs and modal lists.
