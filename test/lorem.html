This example represents a simple web page with the minimum of Awac specific Javascript. All that has been added is a call to start displaying the page triggered in the page's onload event:
(new Awac()).startPage();
If the page has a title element, this is displayed in the action bar.
Navigation away from the page is via the back button.

Awac is designed to make it easy to include simple web pages with only minor adjustments. The sample page (see below) includes a Awac JS file and calls a Awac function when the page loads. Navigation out of the page is by using the back button. Links in the page open in a browser.
