This page demonstrates opening a page In this example we create a global variable to represent Awac called _awac. The variable name must be supplied to the constructor to receive events.
       _awac = new Awac('_awac');
       _awac.setOnPageClose(function(tag,ok,value){
         ...
       });

       _awac.openPage('cities','cities.html',{'name':'Jabba'});

Awac maintains a stack of open pages. When one page opens another, it is pushed onto the stack. When a page ends it is popped off the stack. In addition, Awac allows parameters to be sent to newly opened pages and returned from pages that have ended.
