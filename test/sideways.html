In the sample pages we create a global variable to represent Awac called _awac.
_awac = new Awac('_awac'); ... _awac.replacePage('wizard','wizard2.html',null,true);
These pages provide some examples of how to use Awac. Each page includes a link to view the page outside of the app.

As well as pushing and popping pages on the stack, Awac allows you to replace the page on the top of the stack with a 'next' or 'previous' page. The sample pages (see below) uses this approach to present a wizard like interface to the user.
